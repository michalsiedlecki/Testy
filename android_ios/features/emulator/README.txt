NightlightEmulator is designed to be used for automated tests.
Its main entry point is device.rb class. See in-code documentation for more details.
Example of usage can be found in unit tests (tests/emu_app_tests.rb and tests/emu_backend_tests.rb) and emulator/emulator.rb.

There is also JSON files based interface for manual testing located in emulator subdir. See emulator/README.txt for more details.
